Headings

Headings in Markdown are handled with Hashtags.

# heading
Renders level 1 heading with text "heading"

## heading
Renders level 2 heading with text "heading"

### heading
Renders level 3 heading with text "heading"

#### heading
Renders level 4 heading with text "heading"

##### heading
Renders level 5 heading with text "heading"

###### heading
Renders level 6 heading with text "heading"

Tables

Tables in Markdown are created by seperating table rows with |. They also require a specification between header and content, which specifies the text allignment within particular colums as well as the number of columns. :---- makes a column left centered, :----: centers it in the middle and ----: centers it to the right.

|Column 1|Column 2|Column 3|
|:----|:----:|----:|
|is|is|is|
|alligned|perfectly|alligned|
|to the right|centered|to the left|

Importance

Text can be made bold by adding ** arround it.

**Glad you're using my cheatsheet!**
Will render "Glad you're using my cheatsheet!" in bold.

New lines

New lines can be created by having 2 spaces before a new line in Markdown text.

Theres gonna be a new line below this
Will create a newline (Remember, there are 2 spaces after the "now")
