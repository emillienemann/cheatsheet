Setup

Name

This data will show up on all your commits, make sure not to make a typo! Setup your name globally like this:

git config --global user.name "[firstname lastname]"

E-Mail

This data will show up on all your commits, make sure not to make a typo! Setup your E-Mail adress globally like this:

git config --global user.email "[e-mail]"

Initialization

Initializing Repositories

You can initialize repositories by typing this while being in your target folder (will create a hidden .git folder):

git init

Cloning Repositories

Cloning allows you to get an entire repository from the web (GitHub for example):

git clone [url]

Staging

Status

You can access your Git staging area and it's contents with:

git status

Adding files

You can add files to your git staging area by typing:

git add [file]

You can also add whole directories with:

git add .

Committing

You can commit like this:

git commit -m "message"

Pushing (Updating a remote repository from yours) works like this.

git push

Pushing (Updating your repository from a remote repository) works like this.

git pull
